# Week-1

# Objectives and milestones


```sh
This weeks learning objectives are as follows:

to familiarise yourselves with your chosen project and discuss its requirements with your client.
to be able to create user stories and acceptance tests for your selected project.
to construct a clickable wireframe as a rapid prototype to demonstrate your understanding of what is required.
```

```sh
At the end of the week, you are expected to produce the following in writing:

briefly outline what your software system will do (i.e. what functionality it offers to its users);
briefly outline typical users (or stakeholders) of each software system.
a draft version of user stories and acceptance tests.
a rapid prototype as a proof of concept.
```
